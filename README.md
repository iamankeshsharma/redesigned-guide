# Shree Krishna Engineerings

[![Vibe Coded](https://img.shields.io/badge/Vibe-Coded-FF69B4?style=for-the-badge&logo=visual-studio-code)](https://github.com/danielrosehill/Is-Vibe-Coded)

A modern, high-performance website for Shree Krishna Engineerings, built with a focus on speed, aesthetics, and user experience.

## ✨ Features

- **🏠 Home:** Dynamic landing page with interactive hero sections and company overview.
- **📝 Blog:** Engineering insights, technical articles, and company updates.
- **ℹ️ About:** Detailed company profile, mission statement, and core values.
- **🎨 Modern Design:** Sleek, responsive UI using Tailwind CSS 4 and Radix UI components.
- **📊 Analytics:** Integrated Google Tag Manager and Analytics for performance tracking.

## 🚀 Tech Stack

- **Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite 6](https://vitejs.dev/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Component Primitives:** [Radix UI](https://www.radix-ui.com/)
- **UI Components:** [Material UI (MUI)](https://mui.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Animations:** [Motion (Framer Motion)](https://motion.dev/)
- **Routing:** [React Router 7](https://reactrouter.com/)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or pnpm

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Configure environment variables:
   Copy `.env.example` to `.env` and update the values:

   ```bash
   cp .env.example .env
   ```

### Development

Run the development server with Hot Module Replacement (HMR):

```bash
npm run dev
```

### Build

Compile the project for production:

```bash
npm run build
```
